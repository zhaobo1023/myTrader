## 2. 建议监控的维度（针对日 K 级别）

对于中短线持仓，监控以下指标的组合效果最好：

- **趋势健康度**：收盘价在 20/60 日均线之上（多头形态）。

- **强度信号 (RPS)**：这是你关注的重点。监控 **RPS(120/250)** 是否处于 90 以上，以及是否有掉头迹象。

- **关键点位偏离度**：

  - 距离 20 日均线的百分比（防止乖离率过大）。
  - 距离近 20 日高点的百分比（观察是否即将突破）。

- **量价配合**：今日成交量是否大于 5 日均量的 1.5 倍（放量确认）。

- 其他需要监控的项目（可选）

  - 成熟的券商和量化机构通常会将指标分为几类进行交叉验证，建议你监控以下维度：

    | **维度**            | **关键指标**       | **监控逻辑 (Alert Logic)**                                 |
    | ------------------- | ------------------ | ---------------------------------------------------------- |
    | **趋势 (Trend)**    | 均线 (MA/EMA)      | 价格突破 20/60/120 日线；多头排列/空头排列。               |
    | **动能 (Momentum)** | MACD, RSI, **RPS** | MACD 金叉/死叉；RSI 超买(>70)或超卖(<30)；RPS 强度掉头。   |
    | **支撑/压力**       | 前高/前低, 整数位  | 价格回调至支撑位不破；放量突破关键阻力位。                 |
    | **量价 (Volume)**   | 成交量放大倍数     | “价升量增”或“缩量回调”的量化定义（如放量 > 5日均量 2倍）。 |
    | **波动率**          | 布林带 (Bollinger) | 触碰上轨（超买区域）或下轨（支撑区域）。                   |
