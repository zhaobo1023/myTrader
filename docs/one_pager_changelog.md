## v1.0 初始版本 (2026-04-12)

### 实现内容
- 数据驱动架构：13 个数据块从 DB 预提取，LLM 只做解读
- A-I 九模块框架：上半页 A-E（结构判断）+ 下半页 F-I（投资判断）
- SSE 流式生成 + 当日缓存去重
- 前端集成到 `/stock` 页面第一个默认 tab

### 已知问题
- 毛利率全部缺失（fetcher 硬编码 None）
- 总市值单位错误（total_mv 已是亿元，代码又除以 10000）
- 净利 CAGR 产生虚数（亏损期无保护）
- 增长质量判断逻辑错误（双负增长仍报"增收不增利"）
- 现金流量表数据缺失（financial_cashflow 仅有 2 只银行）
- 股票代码后缀未自动检测（bare code 无法查 K线/RPS/质量因子）
- 通用 prompt 无行业适配（银行用制造业框架分析，水土不服）

## v1.2 通用模板行业适配 (2026-04-13)

### 改动思路
不改 A-I 九模块结构，不改行业路由，在 system prompt 中加入行业自适应规则，让 LLM 根据行业分类调整分析侧重点。

### System Prompt 新增内容

**行业自适应规则**（覆盖 10 个行业大类）：

| 行业 | 分析重心 | PE 使用注意 |
|------|---------|------------|
| 银行/非银金融 | NIM、NPL、拨备、CAR、PB-ROE、股息率 | 受拨备影响大，优先用 PB |
| 食品饮料/消费 | 毛利率、渠道、费用率、品牌溢价 | ROE>15% 是常态 |
| 医药生物 | 研发费用率、管线、集采 | 利润波动大是常态 |
| 房地产 | 土储、合同负债、三道红线 | PE 可能为负，PB<1 常态 |
| 制造业/机械/汽车 | 产能利用率、资本开支、应收 | 周期性强 |
| 电力设备/新能源 | 产能周期、毛利率、政策依赖 | -- |
| TMT | 研发、用户指标、商誉 | 轻资产高估值常态 |
| 周期类（煤/钢/有色/化工） | 价格周期、成本曲线 | PE 顶部低底部高 |
| 公用事业/交运/环保 | 现金流、股息率 | 成长性低但稳定 |

### Part1/Part2 模块级适配

- **B 商业机制**：每个行业大类给出 2-3 个重点分析指标指引
- **F 价值引力**：金融类高负债率不报风险，周期类看周期位置，消费/TMT 看现金流
- **G 人性周期**：周期类不用 PE 判断贵贱，银行优先 PB-ROE，高成长看 PEG
- 所有判词选项新增 "(数据不足，暂无法判断)" 兜底

### 数据层改进
- `_collect_valuation` 新增 PB 分位判断（之前只有 PE 分位）
- PE 为负时标注"分位参考价值有限，建议结合 PB 判断"
