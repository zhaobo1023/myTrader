## 一、核心投资哲学

### 1.1 配置先行，选基第二

唐君的FOF管理思路与传统FOF有本质区别：

| 维度 | 传统FOF | 唐君的FOF |
|------|---------|-----------|
| 起点 | 从选基金出发 | 从配置框架出发 |
| 逻辑 | 选好的基金经理 | 先有观点，再选能实现观点且有α的基金 |
| 风险 | 容易追风格趋势 | 主动暴露已知风险 |

**核心问题**：如果没有配置框架，2021年初你会选出抱团核心资产的基金（因为过去3年表现最好），结果风格切换后大亏。纯粹的基金研究无法解决这个问题。

### 1.2 战略配置 vs 战术配置

| 维度 | 战略配置 | 战术配置 |
|------|----------|----------|
| 周期 | 1-2年或更长 | 月度/季度 |
| 头寸 | 占风险预算的2/3以上 | 不超过1成 |
| 难度 | 相对可把握 | 更难，锦上添花 |
| 内容 | 股债商大类、黄金等 | 行业、风格偏离 |
| 确定性 | 较高 | 较低 |

**关键洞察**：战术配置是"体力活"，需要大量中观研究积累，但贡献有限；战略配置是核心，决定了组合的主要收益来源。

## 二、核心分析框架：货币-信用框架

### 2.1 为什么美林时钟失效？

**有效期**：1980年代至2008年

**失效原因**：
- QE（量化宽松）直接干预市场
- 大财政直接刺激经济
- 政府干预打破了"人性驱动的经济周期"

**本质**：美林时钟基于自由市场经济下的自然周期波动，但2008年后政府成为经济周期的主要驱动力。

### 2.2 货币-信用框架的核心逻辑

```
货币政策（利率、QE等）
        ↓
    影响货币乘数
        ↓
信用创造（居民/企业/政府加杠杆）
        ↓
    决定大类资产表现
```

**核心区分**：
- **货币**：央行的货币政策姿态（宽松/收紧）
- **信用**：实体经济的信用扩张程度（强/弱）

两者可以分离！货币宽松不一定带来信用扩张（如2023-2024中国）。

### 2.3 四象限状态及资产配置策略

| 货币 | 信用 | 资产偏好 | 典型时期 |
|------|------|----------|----------|
| 宽松 | 扩张弱 | 债券、黄金、金融属性强的资产（题材股、比特币） | 2023-2024中国 |
| 宽松 | 扩张强 | 大宗商品、顺周期股票、大盘蓝筹 | 2021美国 |
| 收紧 | 扩张强 | 类滞胀，持有现金 | 短暂过渡阶段 |
| 收紧 | 扩张弱 | 衰退，债券最优 | - |

**关键洞察**：
- 货币宽松但信用扩张受阻时，有利于金融属性强的资产
- 信用扩张起来后，有利于实体经济相关资产

### 2.4 信用创造的三个主体

| 主体 | 观测指标 | 难度 |
|------|----------|------|
| 居民 | 房贷余额变化 | 易观测 |
| 企业 | 信贷、债券发行 | 中等 |
| 政府 | 中央财政 + 地方政府隐性债务 | 最难跟踪 |

**关键洞察**：地方政府隐性债务是最难跟踪但影响巨大的变量。需要定性判断来弥补数据不足。

## 三、具体投资方法论

### 3.1 预期差框架（战术层面核心）

```
预期差 = 基本面变化 - 市场预期
```

**正向预期差**：基本面改善 + 市场预期低位 → 参与
**负向预期差**：基本面走弱 + 市场预期高位 → 回避

#### 刻画预期的指标

| 指标 | 说明 |
|------|------|
| 成交额占比 | 该行业/风格成交额占全市场比例 |
| 融资余额占比 | 融资余额的行业分布 |
| 历史分位数 | 当前值在历史中的位置 |

#### 案例：电解铝

1. 监控铝价与煤炭价差（成本端）
2. 发现价差扩大，利润应改善
3. 但卖方一致预期给零增长
4. 配置后中报业绩+35%，股价大涨

### 3.2 风险预算方法

**核心原则**：
- 每类资产的风险预算 = 自身波动率 × 与组合的相关性
- 负相关资产可以配更多（如黄金与A股）
- 目标：最大回撤不超过5%（常规不超过3%）

**避免受迫交易**：
- 在风险预算阶段就考虑承受能力
- 不要下注过重导致被迫止损
- 止损是最后手段，通常很不划算

**案例**：
> "我看好黄金，为什么不干到20%？因为如果黄金回撤15%，我整个组合最大回撤5%的目标就守不住了，我会被迫止损。而长期我还是看好黄金的，止损后它又涨回去，我就吃大亏了。"

### 3.3 选基金的方法

**两步法**：

1. **界定β暴露**：用7-10个风格因子做逐步回归，确定基金的风格特征
2. **评估纯α**：回归残差的长期均值是否为正

**选择标准**：
- β要符合当前配置观点
- α要为正（否则用ETF更好）
- 场内优先（流动性、费率）

## 四、大类资产观点框架

### 4.1 黄金

**定价框架转变**：
- 旧框架：资产属性（美元实际利率的反向）
- 新框架：货币属性（美元信用的反向）

**核心跟踪指标**：
- 美联储独立性
- 美元武器化程度
- 稳定币政策

**配置策略**：
- 战略底仓：10%，能扛30%回调
- 战术仓位：5%，根据拥挤度调整

**拥挤度指标**：全球主要黄金ETF份额变化

### 4.2 A股

**分类思路**：

| 类型 | 代表 | 核心驱动 |
|------|------|----------|
| 金融属性强 | 题材、科技、成长 | 资金面、利率 |
| 实体经济强 | 大盘蓝筹、顺周期 | 信用扩张 |

**增量资金判断**：

| 资金来源 | 偏好风格 |
|----------|----------|
| 外资 | 核心资产 |
| 公募 | 抱团股 |
| 散户 | 小市值、成长、题材 |
| 国家队/保险 | 红利 |

### 4.3 港股

**双因子驱动**：
- **盈利**：跟大陆经济相关
- **流动性**：跟美元流动性相关（联系汇率制度）

**配置价值判断**：美元流动性改善 + 国内信用扩张边际改善 → 看好

### 4.4 美股

| 指数 | 特点 | 观点 |
|------|------|------|
| 纳斯达克 | 高估值靠AI预期支撑，风险溢价为负 | 谨慎 |
| 标普500 | 被科技股拉高 | 相对更敢配 |
| 标普油气 | 传统行业，与油价正相关 | 关注 |

### 4.5 美债

| 期限 | 定价因素 | 确定性 |
|------|----------|--------|
| 短端 | 政策利率 | 高 |
| 长端 | 市场交易力量 | 中 |

**异常信号**：美债利率上 + 美元下 = 美元吸引力下降（历史罕见）

## 五、方法论精髓

### 5.1 不用长逻辑解释短变化

> "逆全球化是慢变量，只能解释通胀中枢上移，不能解释21-22年通胀飙升。"

**启示**：区分长周期因素和短周期因素，不要混淆。

### 5.2 概率思维

**核心观点**：
- 不以单次结果否定框架
- 越聪明的人越容易从单次结果过度总结

**打麻将例子**：
> 你一对六桶，只能胡六桶。抓个五桶，把六桶打掉，可以胡四七桶。这个决策是对的，因为提升了概率。结果反手摸个六桶自摸了——你不应该后悔，因为决策没错，只是发生了小概率事件。

### 5.3 多维度分散

**分散的层次**：
1. 资产分散（股债商）
2. 策略分散（宏观判断、风格判断、行业判断）
3. 时间分散（不同周期的策略）

**关键洞察**：错一个维度不要紧，多个维度叠加提升胜率。

### 5.4 逻辑必须能解释

**原则**：
- 统计规律必须有逻辑支撑
- 宏观样本少，纯归纳不可靠
- 逻辑可以是基本面、资金面、行为金融

> "我们做了一些研究不是说能解决所有问题、百分之百准确。我们做那么多研究也只是某些阶段把握住了里面的一些，或者说有些是把握住他的主要维度，能把获得收益的概率提升一些。"

## 六、指标体系概览

### 6.1 宏观层（约30个指标）

| 维度 | 指标 |
|------|------|
| 经济产出 | 出口、固定资产投资、消费 |
| 通胀 | PPI、CPI及分项 |
| 政策-货币 | M2、社融、利率曲线 |
| 政策-财政 | 赤字率、国债发行 |

### 6.2 中观层

| 类型 | 指标 |
|------|------|
| 行业业绩领先指标 | 主营产品价格（监控1000+种） |
| 供需指标 | 产能、库存周期 |
| 价差指标 | 电解铝-煤炭、钢铁-铁矿石等 |

### 6.3 资金面

| 类型 | 指标 |
|------|------|
| 散户热情 | 融资余额、新增开户 |
| 增量资金来源 | 北上、公募发行、国家队 |
| 拥挤度 | 成交额占比、ETF份额 |

## 七、框架演进历程

唐君的框架经历了多次迭代：

| 版本 | 方法 | 问题 |
|------|------|------|
| V1 | 30个宏观指标直接打分 | 权重不稳定，样本少 |
| V2 | 指标归类到三维度（政策/经济/通胀） | 有改进但仍不够 |
| V3 | 货币-信用框架 + 定性判断 | 当前使用 |

**启示**：宏观层面样本少，纯量化不可靠，需要逻辑框架 + 定性判断。
