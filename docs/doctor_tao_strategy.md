技术实现方案 & 验证闭环指南

**一、策略原理与理论基础**

**1.1** **两大核心效应**

本系统基于行为金融学中两个经过实证检验的市场异象，由 DeBondt & Thaler (1985) 及 Jegadeesh & Titman (1993) 分别提出。



| **效应**     | **反转效应（****Reversal****）**           | **动量效应（****Momentum****）**           |
| ------------ | ------------------------------------------ | ------------------------------------------ |
| **理论来源** | DeBondt & Thaler 1985                      | Jegadeesh & Titman 1993                    |
| **周期**     | 3～5 年长周期                              | 3～12 个月中周期                           |
| **核心逻辑** | 历史输家未来反转跑赢，历史赢家均值回归     | 近期强势股未来继续强势，弱势股持续弱       |
| **操作含义** | 在主升浪结束后找下一批底部积累中的股票     | RPS稳定在95+的股票参与其中线主升浪         |
| **量化指标** | 价格处于历史低位分位（<30%）+ 近期放量启动 | 欧奈尔RPS（250日涨幅全市场排名百分位）≥ 95 |



**1.2** **两效应的配合时序**

这是原文最有价值的操作逻辑：两类效应不是独立使用，而是形成「接力链」。



| 📌 **接力链逻辑**                                             |
| ------------------------------------------------------------ |
| ① 第一批股票出现主升浪（动量效应确认，RPS≥95，月线连续创新高） |
| ② 同期，第二批股票开始底部积累（反转效应酝酿，历史低位+放量） |
| ③ 第一批冲顶结束后，第二批 RPS 开始快速提升并稳定在 95+      |
| ④ 第二批经过 N 个月调整整理（回调不破底）                    |
| ⑤ 第二批开始新的主升浪，同时第三批开始底部积累               |
| ⑥ 如此循环……这就是板块轮动的本质                             |



**1.3** **关键量化指标定义**

**RPS****（****Relative Price Strength****）****—** **欧奈尔一年期相对强度**

过去 250 个交易日涨幅，在当日全市场 A 股中的排名百分位（0～100）。

- RPS ≥ 95：动量极强，主升浪候选
- RPS 90～95：强势观察区
- RPS < 50：弱势，不参与



**价格历史分位（****Price Percentile Rank****）**

过去 N 年（默认 3 年/750 日）收盘价的历史百分位。

- < 30%：历史低位，可能是反转候选
- \> 70%：历史高位，动量区间



**动量加速度（****RPS Slope****）**

近 4～8 周 RPS 变化的线性斜率，标准化为 Z-Score。

- Z > 1.5：RPS 快速拉升，趋势启动信号
- Z < -1.5：RPS 快速下滑，退出信号

**三、两阶段漏斗筛选逻辑**

选股器的核心是一个两阶段漏斗，将全市场 5000+ 只股票逐步收窄到 10～30 只候选标的。



**3.1** **前置过滤（基本面底线）**

在计算任何技术指标前，先用以下规则剔除不参与的股票，减少噪音。



| **过滤条件**    | **参数**                | **说明**                    |
| --------------- | ----------------------- | --------------------------- |
| 非ST / 退市风险 | 排除名称含ST/*ST        | 避免政策风险和流动性风险    |
| 流动性过滤      | 近60日均成交额 ≥ 5000万 | 小票RPS失真，且实际难以买入 |
| 基本面底线      | 近2年至少1年净利润为正  | 排除纯概念壳股，避免垃圾股  |
| 上市时间        | 上市满1年（250交易日）  | 保证RPS计算有足够历史数据   |
| 价格下限        | 收盘价 ≥ 3元            | 排除仙股，防止统计失真      |



**3.2** **第一层筛选：动量股（买入候选）**

捕捉当前 RPS 已经强势、月线趋势向上、可以跟进参与主升浪的股票。



| **指标**           | **阈值**                     | **量化实现**                                |
| ------------------ | ---------------------------- | ------------------------------------------- |
| RPS（250日）       | ≥ 95，且连续4周≥90           | 过去250日涨幅在全市场排名，周频采样连续达标 |
| 股价站上月线       | 收盘 > MA20（月线）          | 用日线MA20代理月线均线                      |
| 股价站上年线       | 收盘 > MA250                 | 过滤趋势仍向下的弱势股                      |
| 近60日涨幅领先市场 | 60日涨幅排名前30%            | 截面动量验证                                |
| 成交量放大         | 近4周均量 > 过去52周均量×1.2 | 资金关注度提升，动量有资金支撑              |



**3.3** **第二层筛选：反转候选股（预埋仓观察名单）**

捕捉处于底部积累、尚未启动但未来可能启动的股票，对应接力链中「第二批」的识别。



| **指标**     | **阈值**                 | **量化实现**                      |
| ------------ | ------------------------ | --------------------------------- |
| 价格历史低位 | 750日历史分位 < 35%      | 3年内相对低位，反转效应的必要条件 |
| 近期开始启动 | 近60日涨幅 > 15%         | 不是死水，已经有资金开始介入      |
| RPS快速提升  | RPS动量斜率 Z > 1.0      | 近4周RPS提升加速，方向确认        |
| 成交量异动   | 近4周均量 > 52周均量×1.5 | 底部放量是反转的早期信号          |
| RPS尚未到顶  | 当前RPS < 90（尚未过热） | 仍处于启动早期，性价比更高        |



**3.4** **大盘条件开关**

| **⚠** **重要：大盘条件优先于个股信号**                       |
| ------------------------------------------------------------ |
| • 沪指站上50日线（MA50）→ 开启信号输出，参与动量股和反转候选 |
| • 沪指在 MA50 与 MA120 之间 → 降低仓位，只关注最强的动量股（RPS≥97） |
| • 沪指跌破120日线（MA120）→ 暂停所有信号，空仓等待           |
| • 大盘条件是系统性风险的第一道防线，个股再强也不对抗趋势     |

**5.2** **信号失效判断标准**

当实盘表现与回测结果出现持续偏差时，需要重新检验假设。



| **触发重新评估的条件（任一满足）**                    |
| ----------------------------------------------------- |
| 连续3个月实盘胜率 < 40%（回测基准 > 55%）             |
| 连续10个信号均亏损                                    |
| 大盘条件开启但信号股平均跑输沪深300超过10%            |
| 某个信号类型（动量 or 反转候选）在连续6个月无有效信号 |



**重新评估步骤**

1. 检查数据质量：akshare 是否有接口变更导致数据异常
2. 检查市场环境变化：当前市场是否处于策略不适用的状态（如全面熊市）
3. 检查参数是否需要更新：用最近2年数据重新跑参数敏感性分析
4. 检查信号逻辑：是否有新的市场结构变化导致原有逻辑失效



**5.3** **策略局限性备忘**



| **局限性**     | **应对方式**                                                 |
| -------------- | ------------------------------------------------------------ |
| 信号滞后性     | RPS是滞后指标，早期信号已经涨了一段。用反转候选提前埋伏，改善性价比 |
| A股政策风险    | 量化信号无法预测政策打压（如教培、游戏）。需手工加行业政策判断层 |
| 全面牛市失效   | 牛市中RPS区分度下降，几乎所有股票都能达标。此时缩小至RPS≥97  |
| 幸存者偏差     | 回测时使用的历史数据可能已经包含偏差，需纳入退市股票数据     |
| 成交量数据质量 | 部分股票存在虚假放量（大宗交易），需结合换手率过滤异常成交量 |

**六、与行业轮动监控的整合**

个股选股器和申万行业轮动监控（已有）形成「行业→个股」的两级漏斗，可以显著提升信号质量。



**6.1** **两级漏斗整合逻辑**

| **两级漏斗操作框架**                                         |
| ------------------------------------------------------------ |
| 【第一级：行业筛选（已有）】                                 |
| → 行业轮动监控输出：当前处于上升趋势的行业（↑信号）+ 未过热（非🔥） |
| → 形成「可参与行业白名单」                                   |
|                                                              |
| 【第二级：个股筛选（本系统）】                               |
| → 在「可参与行业白名单」内的股票中，运行 RPS + 反转候选筛选  |
| → 行业强势 + 个股强势 = 双重确认，胜率更高                   |
|                                                              |
| 【整合优先级规则】                                           |
| → 优先：行业处于强势区（行业250日分位 > 60）+ 个股动量（RPS≥95） |
| → 其次：行业处于启动早期（行业动量斜率Z>1.5）+ 个股反转候选  |
| → 排除：行业过热（🔥）中的个股，即使个股RPS再高也不追入       |



**6.2** **整合实现步骤**

1. 在 run_screener() 中增加参数 allowed_industries（行业白名单列表）
2. 从行业轮动监控读取当周「↑连续上升且未过热」的行业列表
3. 为每只候选股票附加申万一级行业标签（akshare.stock_individual_info_em）
4. 在周报中新增「行业归属」列，并按行业强弱排序候选股
5. 在行业轮动周报中，增加「各行业内部Top3动量股」的快速展示



**6.3** **整合后的周报结构**



| **周报模块**   | **内容**                                            |
