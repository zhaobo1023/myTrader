### 为什么 hold_days=1 全面亏损

hold_days=1 时每天全换仓，年化交易成本计算：
- 单次买入成本：佣金 0.03% + 滑点 0.1% = 0.13%
- 单次卖出成本：佣金 0.03% + 印花税 0.1% + 滑点 0.1% = 0.23%
- 单次完整交易成本：0.13% + 0.23% = 0.36%
- 年化（250 交易日）：0.36% x 250 = **90%**

即使选股有正 alpha，也被交易成本完全吞噬。这是策略工程问题，不是因子失效。

## 财务风险过滤效果

### 风控过滤规则

```
1. 排除最近年报净利润 <= 0（亏损股）
2. 排除资产负债率 > 70%（高负债）
3. 排除经营现金流 <= 0（造血能力差）
4. 遵循 PIT 规则（1-4月用前年报，5-12月用上年报）
```

### 风控 vs 无风控对比

| hold_days | 无风控年化 | 风控年化 | 差异 | 无风控盈亏比 | 风控盈亏比 | 差异 |
|-----------|-----------|---------|------|-------------|-----------|------|
| 5 | +22.53% | +18.30% | -4.23% | 1.11 | 1.03 | -0.08 |
| 10 | +29.33% | +32.16% | **+2.83%** | 1.20 | 1.21 | +0.01 |
| 20 | +36.40% | +39.85% | **+3.45%** | 1.22 | **1.44** | **+0.22** |

### 分析

- **短持有期（h5）**：风控过滤反而降低收益。被排除的"垃圾股"短期反弹弹性大，5 天内兑现后全身而退
- **长持有期（h10/h20）**：风控过滤显著提升收益和盈亏比。持仓 20 天期间"雷区"股暴雷概率大增，过滤掉亏损+高负债+负现金流的股票有效防雷
- **盈亏比跃升**：h20 场景下盈亏比从 1.22 提升至 1.44（+18%），说明风控的价值在于减少大亏损交易

## 问题诊断与修复

### v2.0 修复（2026-04-12）

#### [RED] factors_cached.py 数据库连接硬编码

**问题**：`factors_cached.py` 中 `load_data()` 直接读取 `ONLINE_DB_*` 环境变量建立 pymysql 连接，
不经过 `config.db.get_connection()`，导致 `DB_ENV=local` 时仍连接线上库。

**修复**：改为使用 `get_connection()`，统一走 DB_ENV 配置。

```python
# 修复前
import os
db_host = os.getenv('ONLINE_DB_HOST', 'localhost')
conn = pymysql.connect(host=db_host, ...)

# 修复后
from config.db import get_connection
conn = get_connection()
```

#### [OK] 回测指标补全

**新增指标**：
- Sortino ratio（仅惩罚下行波动）
- Calmar ratio（年化收益 / 最大回撤绝对值）
- 盈亏比（平均盈利 / 平均亏损绝对值）
- 平均盈利 / 平均亏损
- 平均持有天数

**Sharpe 修正**：原始代码使用 `mean(daily_return) / std(daily_return)`，未扣除无风险利率。
修正为 `mean(daily_return - rf_daily) / std(daily_return, ddof=1)`，rf = 3% 年化。

#### [OK] 财务风险过滤功能

新增 `--exclude-risk` CLI 参数和对应的 universe 过滤逻辑：
- 在 `universe.py` 中增加财务数据查询（JOIN trade_stock_financial）
- 在 `config.py` 中增加 `exclude_risk`, `max_debt_ratio`, `require_positive_profit`, `require_positive_cashflow`
- 在 `backtest.py` `_select_stocks()` 中透传风控参数
- 遵循 PIT 规则：1-4 月使用前年年报，5-12 月使用上年年报

### C. 关键指标说明

| 指标 | 定义 | 优秀 | 良好 | 不佳 | 当前最优 |
|------|------|------|------|------|----------|
| Sharpe | (年化收益 - Rf) / 波动率 | >1.0 | 0.5~1.0 | <0.5 | 0.95 |
| Sortino | (年化收益 - Rf) / 下行波动率 | >1.5 | 0.5~1.5 | <0.5 | 1.02 |
| Calmar | 年化收益 / abs(最大回撤) | >1.0 | 0.5~1.0 | <0.5 | 0.83 |
| 最大回撤 | 净值从峰值到谷底的跌幅 | <10% | 10~30% | >30% | -47.83% |
